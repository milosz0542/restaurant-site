# Restaurant site
Restaurant site is project for PAI classes at University of Wroc≈Çaw.

## Technologies
node.js - as basic API holder. <br>
sql.js - for database connection. <br>
sqlite - as database.

## Setup
To run this project, install it locally using npm:

```
$ cd ../restaurant-site
$ npm install
```

and run with

```
$ npm start
```
